METHODOLOGY
================================================================================

Testing approach: Non-invasive HTTP scanning + HTML parsing
  - Used httpx (no Playwright overhead for initial scans)
  - BeautifulSoup for HTML parsing
  - Python 3.11 on Windows 11
  - Standard browser User-Agent headers (no deception)

No automated abuse:
  - Read-only GET requests
  - Respected robots.txt
  - No brute-force, no CAPTCHA solving
  - 1-second delay between requests

Cloudflare testing:
  - Firefox + Playwright + playwright-stealth library
  - Result: CF still blocks headless access
  - Conclusion: Not cost-effective to pursue
